# E15.2 — UStVA mit Umsatzsteuer (Erlöse + §13b) — Design & Plan

**Status:** Plan. Parent: [[E15 spec]] `docs/superpowers/specs/2026-06-23-erloese-ausgangsrechnungen-design.md`. Branch `feat/e15-2-ustva`.

## Ziel

Die UStVA zeigt heute nur die abziehbare **Vorsteuer** (Soll auf 1576/1571). E15.1 bucht jetzt Erlöse mit **Umsatzsteuer** (Haben auf 1776). E15.2 erweitert die UStVA um die **geschuldete Umsatzsteuer** und die **Zahllast = USt − Vorsteuer**. §13b Reverse-Charge wird beidseitig ausgewiesen (USt 1787 geschuldet + Vorsteuer 1577 abziehbar) — istok-bestätigt; behebt zugleich, dass 1577 bisher gar nicht als Vorsteuer zählte.

## Design

**Konten-Quellen:**
- Vorsteuer (Soll-Seite): `rules.VorsteuerKonten` (1576/1571) + `rules.Rule("reverse_charge").KontoVStRC` (1577) am `RcSatz`.
- Umsatzsteuer (Haben-Seite): `rules.UmsatzsteuerKonten` (1776) + `rules.Rule("reverse_charge").KontoUStRC` (1787) am `RcSatz`.

**Algorithmus (`ComputeUStVA`):** über alle Buchungs-Entries; `Soll`-Entries auf einem Vorsteuer-Konto → Vorsteuer-Summe je Satz; `Haben`-Entries auf einem Umsatzsteuer-Konto → Umsatzsteuer-Summe je Satz. Andere Konten (Aufwand, Erlös, Zahlung) werden ignoriert (nicht in den Maps). `Zahllast = UmsatzsteuerGesamt − VorsteuerGesamt` (negativ = Überschuss/Erstattung).

**Struct-Umbau** (`internal/core/ustva.go`):
```go
type UStVAZeile struct { Satz int; Konto int; Betrag float64 }
type UStVA struct {
    Umsatzsteuer       []UStVAZeile
    Vorsteuer          []UStVAZeile
    UmsatzsteuerGesamt float64
    VorsteuerGesamt    float64
    Zahllast           float64
}
```
(`UStVAZeile.Vorsteuer`→`Betrag`; `UStVA.Zeilen`→`Vorsteuer` + neuer `Umsatzsteuer`.)

**View** (`internal/ui/ustvaview.go`): zwei Abschnitte — „Umsatzsteuer (geschuldet)" und „Vorsteuer (abziehbar)", je Zeilen `Satz % (Konto) Betrag` + Summe — und unten fett die **Zahllast** (bzw. „Überschuss" bei negativ). Monats/Jahr-Umschalter bleibt.

**i18n** (`assets/i18n/de.json`,`en.json`): `ustva.umsatzsteuer`, `ustva.zahllast`, `ustva.ueberschuss`, `ustva.umsatzsteuer.total`/`ustva.vorsteuer.total` (Summen-Label je Block).
