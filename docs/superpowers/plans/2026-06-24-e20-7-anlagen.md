### Task 1: core Asset model + AfA + Anlagenspiegel + PDF

**Files:** `internal/core/anlagen.go` (new), `internal/core/anlagen_test.go` (new), `internal/core/pdfreport.go` (append builder).

**Interfaces:**
```go
type Asset struct { ID string; Bezeichnung string; Anschaffungsdatum string; Anschaffungswert float64; NutzungsdauerJahre int; Konto int; AfaKonto int }
func LoadAssets(path string) ([]Asset, error)
func SaveAssets(path string, a []Asset) error
func IsGWG(a Asset) bool                       // Anschaffungswert <= 800 (net) → Sofortabschreibung
func LinearAfA(a Asset, jahr int) float64      // pro-rata-temporis im Anschaffungsjahr (Monate ab Anschaffungsmonat/12), volle Jahresrate danach, Restbuchwert im letzten Jahr; 0 vor Anschaffung und nach voller Abschreibung; GWG → voller Wert im Anschaffungsjahr
func Restbuchwert(a Asset, jahr int) float64   // Anschaffungswert minus kumulierte AfA bis inkl. jahr (>=0)
type AnlagenRow struct { Asset Asset; AfaJahr, Restbuchwert float64; GWG bool }
func Anlagenspiegel(assets []Asset, jahr int) []AnlagenRow
```

- [ ] **Step 1: test**: asset 1200 € net, ND 3 Jahre, Anschaffung 01.07.2024 → LinearAfA 2024 = 1200/3 * 6/12 = 200; 2025 = 400; 2026 = 400; 2027 = 200 (Rest); 2028 = 0. Restbuchwert 2024 = 1000. A GWG (700 €) → LinearAfA(Anschaffungsjahr)=700, danach 0. Run → fail.
- [ ] **Step 2:** implement `anlagen.go` (Load/Save mirror kassenbuch; AfA math). Run → pass.
- [ ] **Step 3:** `BuildAnlagenspiegelPDF(rows []AnlagenRow, jahr int, title string)` — cols Bezeichnung · Anschaffung · AK · ND · AfA(Jahr) · Restbuchwert · GWG, with totals. Tiny %PDF test. Run → pass + full core.
- [ ] **Step 4:** Commit `E20.7: Asset model + LinearAfA/GWG + Anlagenspiegel + PDF`.
