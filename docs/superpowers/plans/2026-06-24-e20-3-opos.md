### Task 1: core OpenItems + PDF

**Files:** `internal/core/opos.go` (new), `internal/core/opos_test.go` (new), `internal/core/pdfreport.go` (append `BuildOpenItemsPDF`).

**Interfaces:**
```go
type OpenItem struct { Belegnummer, Rechnungsnummer, Datum, Partner string; Betrag float64; AgeDays int; Bucket string }
type OpenItems struct { Forderungen, Verbindlichkeiten []OpenItem; ForderungenGesamt, VerbindlichkeitenGesamt float64 }
func ComputeOpenItems(rows []CSVRow, asOf time.Time) OpenItems
```
Rule: an item is OPEN when `strings.TrimSpace(Bezahldatum)=="" && strings.TrimSpace(BuchungRef)==""`. Open + `Ausgangsrechnung` → Forderung; open + not Ausgangsrechnung → Verbindlichkeit. `Betrag` = Bruttobetrag. `AgeDays` = days from Rechnungsdatum to asOf (0 if unparseable). `Bucket` = "0–30"/"31–60"/"61–90"/">90". Partner = Auftraggeber.

- [ ] **Step 1: test**: rows = one unpaid Ausgangsrechnung (Rechnungsdatum 40 days before asOf), one unpaid expense (10 days), one PAID expense (Bezahldatum set). Assert: 1 Forderung (bucket "31–60"), 1 Verbindlichkeit (bucket "0–30"), paid one excluded; totals correct. Run → fail.
- [ ] **Step 2:** implement `ComputeOpenItems` in opos.go. Run → pass.
- [ ] **Step 3:** `BuildOpenItemsPDF(oi OpenItems, title string) ([]byte,error)` in pdfreport.go — two sections (Debitoren/Kreditoren), columns Belegnr · Datum · Partner · Betrag · Alter(Tage) · Bucket, with section totals. Append a tiny `TestBuildOpenItemsPDF` (PDF starts with %PDF, empty ok). Run → pass + full core.
- [ ] **Step 4:** Commit `E20.3: ComputeOpenItems (aging) + BuildOpenItemsPDF`.
