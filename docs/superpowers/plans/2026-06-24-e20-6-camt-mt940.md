### Task 1: CAMT.053 + MT940 parsers (core)

**Files:** `internal/core/bankimport.go` (new), `internal/core/bankimport_test.go` (new).

**Interfaces:**
```go
func DetectBankFormat(data []byte) string  // "camt" | "mt940" | ""  (camt: contains "<Document" and "BkToCstmrStmt"; mt940: contains ":61:" )
func ParseCAMT053(data []byte) ([]StatementBooking, error)
func ParseMT940(data []byte) ([]StatementBooking, error)
```
- **CAMT.053:** parse `Document/BkToCstmrStmt/Stmt/Ntry` (use `encoding/xml` with structs; ignore namespaces via `xml:"Ntry"` element-name matching). Per `Ntry`: `Amt` (text, parse float; `Ccy` attr ignored), `CdtDbtInd` ("CRDT"→IstGutschrift=true, "DBIT"→false), date from `BookgDt/Dt` (YYYY-MM-DD → reformat to DD.MM.YYYY) else `ValDt/Dt`, text from `AddtlNtryInf` or joined `NtryDtls/TxDtls/RmtInf/Ustrd`. LineIdx = running index, Page=0.
- **MT940:** scan lines; `:61:` = statement line — value date `YYMMDD` (positions after the tag), debit/credit mark `C`/`D` (or `RC`/`RD`), amount (comma decimal, e.g. `1234,56`); the following `:86:` line(s) = purpose/text. Build one StatementBooking per `:61:` (Date from YYMMDD → DD.MM.20YY, Betrag=abs amount, IstGutschrift = mark starts with 'C', Text from the matching :86:).

- [ ] **Step 1: tests** with a small inline CAMT.053 XML (2 Ntry: one CRDT 300,00 dated 2026-01-12, one DBIT 55,24) and a small MT940 (`:61:2601100612...` lines + `:86:` texts). Assert correct count, dates (DD.MM.YYYY), Betrag, IstGutschrift, Text. `DetectBankFormat` returns the right tag. Run → fail.
- [ ] **Step 2:** implement. Run → pass + full core.
- [ ] **Step 3:** Commit `E20.6: CAMT.053 + MT940 parsers + DetectBankFormat`.
